About DistroVitals

DistroVitals tracks the health and vitality of Linux distributions by analyzing:

  • Development Activity - Commits, contributors, and code velocity
  • Community Engagement - Stars, forks, and community growth
  • Maintenance Health - Issue resolution, PR handling, and release cadence

Data is collected from GitHub repositories and other public sources. Scores range from 0-100.
